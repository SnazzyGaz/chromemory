# Chromemory

A fun and challenging color memory test.  
How precisely can you remember and recreate colors?

![Chromemory Preview](https://raw.githubusercontent.com/SnazzyGaz/chromemory/refs/heads/main/icon-512.png)

## 🎮 How to Play

1. Choose your difficulty (Easy / Medium / Hard)
2. Enter a seed (or leave blank for random)
3. A color will be shown for a few seconds — memorize it!
4. Use the HSB sliders to recreate the color as accurately as possible
5. Complete 5 rounds and see your average score

Every game is defined by a **seed** (e.g. `ANVIL-7823`). Share the seed with friends to challenge them on the exact same colors!

## ✨ Features

- Fully offline-capable Progressive Web App (PWA)
- Installable on Android and iOS
- Works great on mobile and desktop
- Precise scoring with perceptual color differences
- Seed system for reproducible games

## 📱 Install as App

**On Android:** Open in Chrome → tap the three dots → "Install app"  
**On iPhone/iPad:** Open in Safari → tap Share → "Add to Home Screen"

## 🛠️ Tech Stack

- Pure HTML, CSS, and JavaScript (no frameworks)
- Service Worker for offline support and caching
- Web App Manifest for PWA features
- Hosted on Netlify
